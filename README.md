# 1. 프로젝트 개요
Kubernetes를 활용한 3Tier 웹 서비스 구축

<br/>
<br/>

# 2. 프로젝트 기획 배경
컨테이너 환경구성( Web : NginX ,WAS : Tomcat , DB : mariadb 사용)

- WEB , WAS , DB 각각 3Tier 인프라 통신
- 부하에 따른 Autoscaling 가능해야 하며 이를 증명
- DB에 대한 정보 변경이 반드시 WAS에서 처리 가능
